Vimrc as Sudo

I’ve only recently discoverd the -E flag for sudo. I keeps the current environment when going into sudo. I find sudo -E vim ... particularly useful since it keeps all my precious vimrc settings when decending into the root regions of my system.

Side note concerning vim: vim can edit remote files with scp://<user>@<host>//<path>, although remember the permissions on the remote file!`
